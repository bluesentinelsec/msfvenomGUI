# msfvenomGUI

msfvenomgui provides a graphical UI over Metasploit's msfvenom tool.

![image](media/msfvenomgui-screenshot.png)

## Prerequisites

- Python 3 Interpreter
- PIP
- msfvenom

## Usage

```bash
# install msfvenomgui with pip
pip install msfvenomgui

# execute msfvenomgui
msfvenomgui
```
